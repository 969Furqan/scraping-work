Advanced Word Processing Applications
Overview
Learn to use advanced MS Word 2016 software features required to prepare complex and prepare professional documents. Learn to manage document options and settings, design advanced documents, create advanced references, and create custom word elements.

What You Need
Microsoft Word 2016 for Windows. You will not be able to complete the course with any other version of Word. Word 365 is not supported in this course, nor is any version of Word running on a Mac.
