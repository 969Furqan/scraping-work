Spatial Database Management and Integration
Overview
Expand on skills learned from Introduction to Relational Database Management Systems (RDBMS) to understand large scale enterprise integration of spatially enabled databases to secure administer, and set rules for data that has a spatial component. Use spatial measurements, functions, and indexes to process points, lines, and polygons. Develop and integrate a Spatial Database Management System (SDBMS) and deploy the database for multiple user access and editing. The course will culminate in students developing an enterprise GIS data model and make use of efficient data processing strategies.
